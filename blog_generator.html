🔑 구글 무료 API 키 발급 방법

1
Google AI Studio 접속
아래 파란 버튼을 클릭해서 Google AI Studio에 접속하세요.
평소 쓰시는 구글(Gmail) 계정으로 로그인하시면 돼요.
2
왼쪽 메뉴에서 API 키 클릭
로그인 후 화면 왼쪽 사이드바에서 Get API key를 클릭하세요.
화면 왼쪽 메뉴 → 🔑 Get API key 클릭
3
새 API 키 만들기
파란색 Create API key 버튼을 클릭하세요.
Create API key in new project를 선택하세요.
+ Create API key 클릭 → Create API key in new project 선택
4
키 복사해서 붙여넣기
AIza로 시작하는 긴 문자열이 나타나요.
복사 후 이 앱의 입력창에 붙여넣기(Ctrl+V) 하세요.
AIzaSy... 옆 복사 아이콘 클릭 → 앱 입력창에 Ctrl+V
✓
완전 무료예요!
카드 등록 없이 하루 500번 무료로 사용 가능해요.

Google AI Studio 바로 가기 →

📷 Unsplash 무료 이미지 키 발급 방법

1
Unsplash 개발자 페이지 접속
아래 버튼을 눌러 Unsplash 개발자 페이지에 접속하세요.
Unsplash 계정이 없으면 무료로 가입하면 돼요.
2
New Application 클릭
로그인 후 Your apps 화면에서 New Application을 클릭하세요.
약관 체크박스 4개 모두 체크 → Accept terms 클릭
3
앱 이름 입력 후 생성
Application name에 아무 이름 입력 → Description에 간단한 설명 입력 → Create application 클릭
4
Access Key 복사해서 붙여넣기
Keys 섹션에서 Access Key를 복사해서 이 앱 입력창에 붙여넣기 하세요.
Access Key 옆 복사 아이콘 클릭 → 앱 입력창에 Ctrl+V
✓
완전 무료예요!
시간당 50회 무료 · 카드 등록 불필요 · 고품질 사진 제공
⚠️ 주의: Access Key는 비밀번호처럼 관리하세요. 타인과 공유하지 마세요.
Unsplash 개발자 페이지 바로 가기 →
